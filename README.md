# Modelagem Matemática Computacional
Nesta disciplina os seguines tópicos foram abordados:
- Etapas do processo de modelagem
- Modelos científicos e definição de modelo matemático
- Modelagem dos problemas com equações diferenciais
- Modelagem dos problemas com otimização e grafos
- Construção de modelos deterministicos
- Construção de modelos probabilísticos utilizando Cadeia de Markov
